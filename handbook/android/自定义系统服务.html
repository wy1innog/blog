    # 自定义系统服务

    Created by: wylin Created time: May 30, 2023 6:37 PM Status: Approved

    # 前言

    常见的AMS、PWS、WMS等等都是系统服务，运行于system_server进程，并且向servicemanager进程注册其Binder以便其他进程获取binder与对应的服务进行通信。为了新增自定义系统服务，我们可以参考AMS等原生系统服务编写如下文件：

    1、AIDL文件：生成Binder类，其中Stub即为Binder的服务端；

    2、XXManagerService：系统服务类，继承自Stub；

    3、XXManager：封装了AIDL接口方法的类，相当于Binder客户端（Proxy），其他进程通过此类完成与系统服务的通信。

    # 增加系统服务

    # 1、定义服务接口AIDL

    首先要定义服务名是什么，提供什么样的接口。在frameworks/base/core/java/com/hello/service中编写IHelloManager.aidl

    // IHelloManager.aidl
package com.hello.service;

// Declare any non-default types here with import statements

interface IHelloManager {
   void sendMessage(String msg);
}

    # 2、实现接口，系统服务

    在frameworks/base/core/java/com/hello/service中创建HelloManagerService.java文件（接口实现），内容如下：

    package com.hello.service;

import android.os.RemoteException;
import android.util.Log;

public class HelloManagerService extends IHelloManager.Stub {

    private static final String TAG = "HelloManagerService";

    @Override
    public void sendMessage(String msg) throws RemoteException {
        Log.i(TAG, "sendMessage: " + msg);
    }
}

    # 3、将服务添加到ServiceManager，客户端代理

    在frameworks/base/core/java/com/hello/service下编写HelloManager.java

    package com.hello.service;

import android.os.RemoteException;
import android.util.Log;

public class HelloManager {

    private static final String TAG = "HelloManager";
    private IHelloManager service;

    public HelloManager(IHelloManager service) {
        this.service = service;
    }

    public void sendMessage(String msg) {
        try {
            service.sendMessage(msg);
        } catch (RemoteException e) {
            e.printStackTrace();
        }
    }
}

    # 6、注册应用系统服务获取器

    在frameworks/base/core/java/android/app/SystemServiceRegistry.java注册服务获取器：

    import com.hello.service.IHelloManager;
import com.hello.service.HelloManager;
static{
    registerService(Context.HELLO_SERVICE, HelloManager.class,
        new CachedServiceFetcher<HelloManager>() {
            @Override
            public HelloManager createService(ContextImpl ctx) throws ServiceNotFoundException  {
                // 获取服务
                IBinder b = ServiceManager.getServiceOrThrow(Context.HELLO_SERVICE);
                IHelloManager service = IHelloManager.Stub.asInterface(b);
                return new HelloManager(service);
            }
        });
}

    # 8、配置SELinux权限

    没有配置权限的话，服务中相关api供外部调用会提示错误

    1. 在文件system/sepolicy/private/service_contexts添加如下内容

      hello                               u:object_r:hello_service:s0
      1

    2. 在文件system/sepolicy/public/service.te添加如下内容:

      type hello_service, app_api_service, ephemeral_app_api_service, system_server_service, service_manager_type;
      1

    3. 在文件system/sepolicy/private/untrusted_app_all.te添加如下内容:

      allow untrusted_app hello_service:service_manager find;
      1

    4. 保持其他文件修改保持一致，如下一个段落文件中的内容必须完全一致，多一行空格都不行

    # 8、配置白名单

    如果以当前案例的方式新增自定义系统服务，因为SystemServiceRegistry 中需要使用到com.hello下的类，为了让其获取此包下类的引用，需要配置：build/core/tasks/check_boot_jars/package_whitelist.txt ，加入：com\\.hello\\..*。否则会因为无法获取类引用而编译报错！

    dalvik\\..*
libcore\\..*
android\\..*
com\\.android\\..*
com\\.hello\\..*

    # 10、在App中使用

    如果是系统应用，可以直接import导入使用，如下方：

    import ...
import com.hello.service.HelloManager;

public class MainActivity extends Activity {
    private static final String TAG = "MainActivity";
    private TextView mTvTxt;
    private Button mBtnClick;

    @Override
    protected void onCreate(@Nullable Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        initView();
    }

    private void initView() {
        mTvTxt = findViewById(R.id.tv_txt);
        mBtnClick = findViewById(R.id.btn_click);
        mBtnClick.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                test();
            }
        });
    }

    private void test() {
        HelloManager myServiceManager = (HelloManager) getSystemService("hello");
        myServiceManager.sendMessage("Hello你好");
        Log.i(TAG, "initData: send Hello");
    }
}
